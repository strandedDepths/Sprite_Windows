# DK Mini Template

This is a **fan-made template project** for creating a window containing sprites animation with custom music in Python using Pygame.

> /!\ **Note:** This repository **does not include any assets nor mp3 files**. 
> Users must provide their **own background images, sprites, and music**.

## Features

- Window setup with dynamic resizing
- Placeholder sections for:
  - Background
  - Sprites
  - Music
- Easy-to-edit `main.py` template for customization

## How to Use

1. Clone this repository
2. Add your own assets in the `assets/` subfolders
3. Run the template:

```bash
python3 main.py
